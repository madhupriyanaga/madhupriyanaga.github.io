# Portfolio Website: Madhupriya Naga

This is my personal portfolio website, showcasing skills, projects, achievements, and contact information. The website is designed to be simple, stylish, and responsive.

## Features

- **About Section**: Brief introduction and background.
- **Skills Section**: Highlights technical skills and tools.
- **Projects Section**: Showcases key projects with descriptions and GitHub links.
- **Achievements Section**: Lists notable achievements.
- **Resume Viewer**: Embedded PDF viewer for the resume with a download option.
- **Certificate Section**: Provides a downloadable certificate for the Meta Hacker Cup 2025.
- **Interactive Elements**: Includes a typewriter effect and a "Magic Joke" button.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout.
- **JavaScript**: For interactivity.
- **Bootstrap**: For responsive design.
- **Font Awesome**: For icons.

## How to Use

1. Clone the repository or download the source code.
2. Open the `index.html` file in any modern web browser.
3. Replace `resume.pdf` with your own file.

## Folder Structure

```
portfolio/
├── index.html       # Main HTML file
├── styles.css       # Custom CSS file
├── script.js        # JavaScript file for interactivity
├── resume.pdf       # Resume file (replace with your own)
```
